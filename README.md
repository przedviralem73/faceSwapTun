# Face Swap Camera 📸

Программа для замены лица на видео с камеры.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

### С изображением Тунтунсахура:

1. Поместите изображение `tuntunsahur.png` в папку с программой
2. Запустите:

```bash
python face_swap.py tuntunsahur.png
```

### Со стандартным смайликом:

```bash
python face_swap.py
```

## Управление

- **Q** или **ESC** - выход из программы
- **S** - сохранить скриншот

## Требования

- Python 3.8+
- OpenCV
- MediaPipe
- Web-камера

## Советы

- Используйте изображения с прозрачным фоном (PNG) для лучшего результата
- Изображение лица должно быть относительно квадратным
- Программа автоматически масштабирует изображение под размер лица
